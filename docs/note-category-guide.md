# noteの記事にカテゴリーを付ける方法

ウェブサイトの「Column」ページでは、noteの記事をカテゴリーで絞り込めるようになっています。
この仕組みは **noteのハッシュタグ** を利用しています。

## やること

noteで記事を公開するとき、**以下のいずれかのタグを1つ付けてください。**

| タグ名（そのまま入力） | どんな記事に付けるか |
|---|---|
| `活動レポート` | イベントや活動の報告 |
| `プロジェクト/メンバー紹介` | メンバーやプロジェクトの紹介記事 |
| `お知らせ` | イベント告知・お知らせ |
| `食文化見聞録` | 食文化に関するコラム・記録 |

## 注意点

- タグは **noteの通常のハッシュタグ機能** でそのまま付けるだけでOKです
- 他のタグ（`#淡路島` など）はこれまで通り自由に付けてください
- 上の4つのうち **1つだけ** 付ければ、サイト上でカテゴリーとして表示されます
- どれも付けなかった場合、記事は表示されますが「カテゴリーなし」の状態になります

## 仕組み（参考）

サイト側では `constants/site.ts` の `NOTE_CATEGORIES` でカテゴリー一覧を定義しています。
note APIから取得した記事のハッシュタグと照合し、一致したものをカテゴリーとして扱います。

カテゴリーを追加・変更する場合は `NOTE_CATEGORIES` を編集してください。
